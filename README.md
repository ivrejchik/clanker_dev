# Good Boss Marketplace

Claude Code plugin marketplace containing the good-boss plugin and related skills.

## Directory Structure

```
.claude-plugin/                     Marketplace manifest
plugins/
  good-boss-marketplace/            The good-boss_doesnot_work plugin
    plugins/good-boss_doesnot_work/
      commands/                     Plugin commands
      skills/boss-delegation-playbook/  Boss delegation skill
  add-dag/                          DAG plugin
  solve/                            Solve plugin
```

## Plugins

- **good-boss_doesnot_work** -- Boss delegation playbook and commands
- **add-dag** -- DAG-related skill
- **solve** -- Problem-solving skill
